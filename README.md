# ネットワークアドレス管理表

ホームネットワークの IP アドレス・サブネット情報を一覧管理するシングルページアプリです。
ビルド不要・依存ライブラリなし。`index.html` をブラウザで開くだけで動作します。

![アプリ画面](./screenshot.png)

---

## 機能

| 機能 | 説明 |
|---|---|
| キーワード検索 | IP / 名称 / 役割 / ネットワークID / 割当 / メモ の全列を対象にリアルタイム絞り込み |
| ネットワークフィルタ | フィルタチップで特定サブネットのホストだけを表示 |
| ソート | 列ヘッダクリックで昇順・降順切替（IP はオクテット数値比較） |
| ホスト追加 | `+ Add Host` ボタンからモーダルフォームで新規登録（IP 重複チェック・バリデーション付き） |
| ホスト編集 | 行の「編集」ボタンでフォームを開き内容を上書き保存 |
| ホスト削除 | 行の「削除」ボタン → 確認ダイアログで削除 |
| サブネット一覧 | VLAN・ゲートウェイ・DHCP 範囲をカード形式で表示 |
| 集計 | Subnets / Hosts / Static の件数をリアルタイムで集計 |

## 使い方

### 閲覧

```
index.html をブラウザで開く
```

ローカルファイルとして直接開けます。サーバー不要です。

### データの編集（永続化）

`index.html` 内の `<script id="ipData" type="application/json">` を直接編集します。

**ホストの追加例:**

```json
{
  "ip": "192.168.5.20",
  "name": "NAS",
  "role": "Storage",
  "network_id": "home-lan",
  "assign": "static",
  "note": "バックアップ用"
}
```

**ネットワークの追加例:**

```json
{
  "id": "office-lan",
  "cidr": "192.168.10.0/24",
  "vlan_id": 10,
  "gateway": "192.168.10.1"
}
```

> UI の追加・編集・削除はメモリ上のみ有効です。ページをリロードすると JSON の初期値に戻ります。
> 恒久的に変更するには上記の JSON 直接編集を行ってください。

詳細は [`ネットワークアドレス管理_データ追加手順.md`](./ネットワークアドレス管理_データ追加手順.md) を参照してください。

## 技術スタック

- **HTML / CSS / JavaScript** — フレームワーク・ビルドツール不使用
- **フォント** — [Syne](https://fonts.google.com/specimen/Syne)（見出し）・[IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono)（データ表示）via Google Fonts
- **データ** — `index.html` に埋め込んだ JSON（外部 API・ストレージなし）

---

## 注意事項

- データの永続化機構（localStorage・バックエンド等）はありません
- UI 操作（追加・編集・削除）はページリロードでリセットされます
- Google Fonts の読み込みにはインターネット接続が必要です（オフライン時はシステムフォントにフォールバックします）
