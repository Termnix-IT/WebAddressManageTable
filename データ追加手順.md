# IP検索ツール データ追加手順

## 1. 編集するファイル
- `index.html`

## 2. 編集する場所
- `index.html` 内の以下ブロックを編集します。
- `<script id="ipData" type="application/json"> ... </script>`

## 3. 追加できるデータ
- 機器情報: `hosts`
- サブネット情報: `networks`（VLAN情報を含む）

## 4. 機器（hosts）の追加方法
- 機器1台ごとに `hosts` 配列へ1オブジェクト追加します。

追加例:

```json
{
  "ip": "192.168.30.20",
  "name": "NAS",
  "role": "storage",
  "network_id": "home-lan",
  "assign": "static",
  "note": "バックアップ用"
}
```

## 5. サブネット（networks）とVLANの追加方法
- サブネット1つごとに `networks` 配列へ1オブジェクト追加します。
- サブネット一覧では以下を表示します。
  - サブネットアドレス: `cidr`
  - VLAN有無/番号: VLAN番号キーの有無で判定

VLAN番号として認識されるキー:
- `vlan_id`
- `vlanId`
- `vlan_number`
- `vlanNumber`
- `vlan.number`
- `vlan.id`

追加例（推奨は `vlan_id`）:

```json
{
  "id": "office-lan",
  "cidr": "192.168.40.0/24",
  "vlan_id": 40,
  "gateway": "192.168.40.1",
  "dhcp": {
    "enabled": true,
    "range": {
      "start": "192.168.40.100",
      "end": "192.168.40.199"
    }
  }
}
```

VLANなしの例:

```json
{
  "id": "guest-lan",
  "cidr": "192.168.50.0/24",
  "gateway": "192.168.50.1"
}
```

## 6. 反映イメージ

```html
<script id="ipData" type="application/json">
{
  "networks": [
    {
      "id": "home-lan",
      "cidr": "192.168.30.0/24",
      "vlan_id": 30,
      "gateway": "192.168.30.1"
    },
    {
      "id": "office-lan",
      "cidr": "192.168.40.0/24",
      "vlan_id": 40,
      "gateway": "192.168.40.1"
    }
  ],
  "hosts": [
    {
      "ip": "192.168.30.10",
      "name": "Archer AX23",
      "role": "wifi-router",
      "network_id": "home-lan",
      "assign": "static",
      "note": "固定IP"
    },
    {
      "ip": "192.168.30.20",
      "name": "NAS",
      "role": "storage",
      "network_id": "home-lan",
      "assign": "static",
      "note": "バックアップ用"
    }
  ]
}
</script>
```

## 7. JSON編集時の注意
- オブジェクト同士の区切りに `,` が必要です。
- 最後の要素の末尾には `,` を付けません。
- 文字列は必ず `"` で囲みます。

## 8. 保存後の反映
- `index.html` を再読み込みすると、サブネット一覧・hosts一覧・検索結果に反映されます。

## 9. 将来拡張（機器属性やVLANアドレス）
- 今は `hosts` を表示・検索対象にしています。
- 将来 `devices` や `vlanAddresses` を追加する場合は、まず `ipData` 内に配列を追加し、その後 `app.js` の `dataModelAdapters` に対象定義を追加します。
